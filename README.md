Game Title: Tense Trail
게임 로직: 
1. 플레이어는 WASD로 이동한다. 
2. Enter 키를 눌러 카드를 선택한다. 
3. WordCard는 자신이 가진 단어 정보를 MazeManager에게 전달한다. 
4. MazeManager는 현재 목표 단어와 비교하여 정답/오답을 판정한다. (플레이어는 take --> took --> taken 순서로 단어카드를 선택해야한다.) 
5. 올바른 단어카드 3개를 선택 후, Exit에 도착하면 게임이 종료된다. 
6. ResultUI는 플레이 시간과 오답 횟수를 표시한다.
